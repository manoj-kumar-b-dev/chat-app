🚀 Chat App - Real-time Messaging Platform

A professional, real-time chat application built with React 19, Firebase, and Vite. Features include instant messaging, Google Authentication, file sharing, and responsive design.

✨ Features

- Real-time Chat: Powered by Firebase Realtime Database.
- Authentication: Secure Login/Register and Google Sign-In.
- File Sharing: Upload images, videos, and PDFs.
- Responsive UI: Beautiful, modern interface using Tailwind CSS and Framer Motion.
- Persistent Sessions: Stay logged in across refreshes.
- User Search: Easily find and start chats with other users.

🛠️ Tech Stack

- Frontend: React 19, Vite, Tailwind CSS, React.
- Backend/Database: Firebase (Auth, Realtime Database, Storage).
- Deployment: GitHub Pages (Frontend).


🔒 Security

- Sensitive keys are stored in `.env` and excluded from Git via `.gitignore`.
- Firebase Security Rules should be configured to restrict data access to authenticated users only.
